# 🎬 Movie Recommendation System

A **console-based Movie Recommendation System** built with **Python**. This application allows users to manage a collection of movies by adding, viewing, searching, updating, deleting, and filtering movies. Movie data is stored in a JSON file, allowing it to persist even after the program is closed.

---

## ✨ Features

* ➕ Add Movie
* 📋 View All Movies
* 🔍 Search Movie
* ❌ Delete Movie
* ✏️ Update Movie Details
* ⭐ View Top Rated Movie
* 🎭 Recommend Movies by Genre
* 🎯 Filter Movies by Minimum Rating
